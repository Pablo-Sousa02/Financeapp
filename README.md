# FinanceApp – Controle Financeiro em Java

Aplicação de controle financeiro desenvolvida em Java, com foco em Programação Orientada a Objetos e boas práticas de organização de código.

## Funcionalidades
- Adicionar receitas
- Adicionar despesas
- Classificação por categorias
- Listagem de transações
- Cálculo automático de saldo
- Validação de entradas

## Conceitos aplicados
- Programação Orientada a Objetos (POO)
- Abstração
- Herança
- Polimorfismo
- Encapsulamento
- Enum
- Collections
- Organização em pacotes
